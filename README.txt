Espace Hassan demo website

Files:
- index.html
- styles.css
- script.js

How to use:
1. Open index.html in your browser.
2. Replace the placeholder visuals with real restaurant images.
3. Update phone numbers, addresses, and branch details if needed.
4. Replace Google Maps placeholder links with real links.
5. Host it on Netlify, Vercel, GitHub Pages, or any basic hosting.

Notes:
- This is a static website. No build step required.
- Designed to be easy to edit and pitch as a restaurant landing page.
- Built as a demo/spec concept before client approval.
